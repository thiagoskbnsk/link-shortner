# Link Shortner

Este é um projeto de encurtador de links desenvolvido como parte dos desafios da pós-graduação. O objetivo principal é criar uma aplicação que permita aos usuários encurtar URLs longas, tornando-as mais fáceis de compartilhar e gerenciar.

## Funcionalidades

- Encurtar URLs longas em links curtos e amigáveis.
- Redirecionamento automático ao acessar o link curto.
- Registro e gerenciamento de links encurtados.
